# Livraisons mensuelles reglementaires

## Informations générales

Afin de simplifier le lancement du script, il est conseillé d'utiliser l'environnement de travail RStudio (https://www.rstudio.com/products/rstudio/download/#download).

## Aides

* Si vous avez des problèmes de connexion avec la base de données, vérifiez  que vous êtes bien connecté au VPN de l'IRD, ainsi que vos paramètres de connexion.

* Pensez à vérifier que vous utilisez bien la dernière version du package toolBox.

* Si vous n'avez pas de fichier de configuration, vous pouvez en créer un facilement en répondant non dans la console de R lors du lancement de la fonction "ConfigurationFile".

* Si l'export semble incohérent, vérifiez que les vues matérielles de la base de données sont bien à jour.

* N'hésitez pas à signaler des bugs ou des possibilités d'améliorations.
